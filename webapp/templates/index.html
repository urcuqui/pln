Resumen

Este desarrollo tiene como finalidad ser un analizador morfologico basado en las APIs de Freeling.
A partir de una sentencia de entrada el sistema realiza un proceso y despliega la informacion
morfologica de cada palabra.
Freeling es una herramienta open source para analisis de lenguaje que hace uso de las etiquetas EAGLES,
entre algunas de sus utilidades empleadas en este proyecto se encuentran:

  • "tokenizer.dat" permite obtener el metodo para la Tokenizacion de palabras sobre el texto segmentado
  • "splitter.dat" luego de tokenizar, esta utilidad permite detectar el limite de una frase

    • http://nlp.lsi.upc.edu/freeling/

    Tecnologias implementadas
  • Python2.7
  • Django
  • Freeling3.1
